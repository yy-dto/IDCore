Definition:Contains domains for act reasons used in clinical research.

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Import all the codes that are contained in ClinicalResearchEventReason
  • Import all the codes that are contained in ClinicalResearchObservationReason
  • Include all codes defined in http://terminology.hl7.org/CodeSystem/v3-ActReason
