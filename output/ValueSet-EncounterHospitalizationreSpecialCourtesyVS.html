ValueSet: Special courtesy

Official URL: https://fhir.kemkes.go.id/r4/ValueSet/EncounterHospitalizationreSpecialCourtesyVS Version: 0.1.0
Active as of 2024-02-15 Computable Name: EncounterHospitalizationreSpecialCourtesyVS

This value set defines a set of codes that can be used to indicate special courtesies provided to the patient.

References

This value set is not used here; it may be used elsewhere (e.g. specifications and/or implementations that use this content)

Logical Definition (CLD)

This value set includes codes based on the following rules:

  • Include all codes defined in http://terminology.hl7.org/CodeSystem/v3-EncounterSpecialCourtesy
  • Include all codes defined in http://terminology.hl7.org/CodeSystem/v3-NullFlavor

 

Expansion

Expansion based on:

  • codesystem EncounterSpecialCourtesy v2.1.0 (CodeSystem)
  • codesystem NullFlavor v2.1.0 (CodeSystem)
