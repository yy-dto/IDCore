CodeSystem: ActStatus

Official URL: http://terminology.hl7.org/CodeSystem/v3-ActStatus Version: 0.1.0
Active as of 2019-03-20 Computable Name: ActStatus
Other Identifiers: OID:2.16.840.1.113883.5.14

Codes representing the defined possible states of an Act, as defined by the Act class state machine.

This Code system is referenced in the content logical definition of the following value sets:

  • ActStatus

Concepts

This case-sensitive code system http://terminology.hl7.org/CodeSystem/v3-ActStatus defines the following codes in a Is-A hierarchy:

LvlCodeDefinitioninternalIdstatus
1 normal Encompasses the expected states of an Act, but excludes "nullified" and "obsolete" which represent unusual terminal states for the life-cycle. 15936 active
2   aborted The Act has been terminated prior to the originally intended completion. 15941 active
2   active The Act can be performed or is being performed 15942 active
2   cancelled The Act has been abandoned before activation. 15938 active
2   completed An Act that has terminated normally after all of its constituents have been performed. 15940 active
2   held An Act that is still in the preparatory stages has been put aside. No action can occur until the Act is released. 15937 active
2   new An Act that is in the preparatory stages and may not yet be acted upon 15943 active
2   suspended An Act that has been activated (actions could or have been performed against it), but has been temporarily disabled. No further action should be taken against it until it is released 15939 active
1 nullified This Act instance was created in error and has been 'removed' and is treated as though it never existed. A record is retained for audit purposes only. 15934 active
1 obsolete This Act instance has been replaced by a new instance. 15935 active
