ValueSet: v3 Code System ActPriority

Official URL: https://fhir.kemkes.go.id/r4/ValueSet/ActPriorityVS Version: 0.1.0
Active as of 2024-02-04 Computable Name: ActPriorityVS

A set of codes (e.g., for routine, emergency), specifying the urgency under which the Act happened, can happen, is happening, is intended to happen, or is requested/demanded to happen.

  • IDCoreEncounter

Logical Definition (CLD)

  • Include all codes defined in http://terminology.hl7.org/CodeSystem/v3-ActPriority
